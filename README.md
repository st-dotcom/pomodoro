# Pomodoro Pro

> **生産性を最大化するための、技術志向なブラウザベース・ポモドーロタイマー**

Pomodoro Proは、集中時間と休憩時間を厳密に管理し、ユーザーの生産性を最適化するためのツールです。単なるタイマー機能を超え、タスク管理、デッドライン通知、および統計データの永続化機能を備えた包括的なワークフロー支援アプリケーションです。

---

## 概要

本プロジェクトは、Web標準技術のみを用いて構築された高機能ポモドーロタイマーです。ユーザーインターフェースの即時性と、Web Storage APIを用いたデータ整合性を両立させています。

---

## 主な機能

### 1. ポモドーロ・タイマー

* **自動セッション移行**: 25分の集中（Work）と5分の休憩（Rest）を自動的にスイッチします。
* **ステータス表示**: 現在の状態（集中/休憩）を視覚的に判別できるカラーインジケーターを搭載。

### 2. タスク管理と実行ログ

* **コンテキスト割り当て**: 各セッションに対して固有のタスク名を付与可能。
* **ログ記録**: セッションの完遂（Completed）および中断（Aborted）を自動記録し、履歴を一覧表示します。

### 3. 統計分析

* **累積データの算出**: 総集中時間のリアルタイム計算。
* **パフォーマンス指標**: セッション完遂率（Completion Rate）を算出し、作業効率を可視化します。

### 4. 期限リマインド機能

* **動的通知**: 任意の期限時刻を設定し、セッション実行中に期限に達した際、即座に通知を行います。
* **マルチレイヤー通知**:
* `Browser Notification API` によるデスクトップ通知。
* フォールバックとしてのブラウザアラート通知。



### 5. データの永続化

* **LocalStorage統合**: `Web Storage API` を活用。ブラウザの終了やリロード後も、累計統計や履歴データが保持される設計になっています。

## セットアップと使用方法

### インストール

本リポジトリをクローンまたはダウンロードしてください。

```bash
git clone https://github.com/username/pomodoro-pro.git
cd pomodoro-pro

```

### 実行方法

1. ブラウザで `index.html` を開きます。
2. タスク名と（任意で）期限時刻を入力します。
3. `Start Session` ボタンをクリックしてタイマーを開始します。
4. 25分経過後、自動的に休憩モードへ移行します。

### データの管理

* 履歴や統計を完全にリセットする場合は、画面下部の **"Clear all data"** を実行してください。

---

## アーキテクチャの詳細

本アプリケーションは、スケーラビリティとメンテナンス性を考慮し、**単一のステートオブジェクト（App.state）** による状態管理を採用しています。

* **UI/Logicの分離**: ロジック層で `App.state` を更新し、その変更をUI層に反映させる単方向のデータフローに近い設計を模倣しています。
* **永続化レイヤー**: ステートが更新されるたびに LocalStorage への同期が行われ、データの不整合を防止しています。
